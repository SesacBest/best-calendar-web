# S-Calendar

## 프로젝트 개요

### 프로젝트 목적과 주요 기능
#### 목적
- Calendar를 활용하여 일정, 할 일, 일기를 기록하는 서비스를 구현한다.
#### 주요기능
1. 로그인 / 회원가입
2. 캘린더
3. 일정 / 할 일 / 일기 기록

### 문제 정의와 타겟
#### 문제
- 많은 캘린더 앱들이 너무 많고 복잡한 기능으로 사용성이 떨어짐
- 필수 기능만 탑재, 심플한 디자인으로 해결책 제시, 솔루션 제안, 서비스 제공
#### 타겟
- 캘린더 앱의 필수 기능만 사용하고 싶은 한국인
- 복잡하지 않은 캘린더 앱을 사용하고 싶은 유저
- 심플하고 깔끔한 캘린더 앱을 사용하고 싶은 누구나
